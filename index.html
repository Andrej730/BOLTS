BOLTS

BOLTS is open library of technical specifications.

Using BOLTS with OpenSCAD

Download the OpenSCAD tarball from the Downloads page and extract it into a folder. Start OpenSCAD from this folder or Adjust your OpenSCAD library path to include BOLTS. Then add this at the top of your scad file:

{% highlight python %} include {% endhighlight %}

You can now use BOLTS to add standard parts like a washer:

{% highlight python %} ISO7089("M4"); {% endhighlight %}

Using BOLTS with FreeCAD

Download the FreeCAD tarball from the Downloads page and extract it into a folder. Start FreeCAD using the supplied script:

{% highlight bash %} ./launch_freecad {% endhighlight %}

This adjusts the module search paths to include the BOLTS widget. The widget can be started by typing into the FreeCAD python console:

{% highlight python %} import freecad_bolts {% endhighlight %}

There you can select a part from the tree, give values for the parameters and add it to the current document.
