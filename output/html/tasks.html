Unsupported licenses

For some classes a unsupported license name and/or url was found. Often this is the result of typos. Check with the list in the specifications to identify these cases. If the license is not on the list, try contact all the copyright holder of the file and ask them to relicense or dual license this file under one of the supported licenses, or remove the file.
