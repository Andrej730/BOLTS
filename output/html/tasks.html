Missing base geometries

Some classes can not be used in one or more CAD packages, because no geometry is available.

Detailed information how to add a geometry can be found here for OpenSCAD, and here and here for FreeCAD.

Missing common parameters

Some classes have no common parameters defined.

This either happens if a class contains non-discrete free parameters or if it explicitly a empty common parameters set is given, in order to suppress output for this class or work around bugs.
