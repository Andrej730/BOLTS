What can I do?

There are loads of things that you can do to help improve BOLTS:

  • make suggestions how to improve the website and the documentation or report problems
  • report or even fix errors in part specifications
  • suggest or add parts that are still missing
  • give feedback on the workflow in the different CAD applications and how to improve it
  • create drawings for the online specification pages
  • dive in the code of bolttools and fix bugs, extend and improve it

BOLTS scans its database for inconsistencies, common problems, missing or unclear informations and automatically generates a list of things that should be fixed or improved. This is a great starting point for contributing to BOLTS

How to do that?

For many of the things suggested above there are detailed guides in the Documentation section.

There are several ways to report problems or make suggestions:

  • Problems with the documentation can be reported via the comments section embedded on each page.

  • For FreeCAD related things, there is a development thread on the FreeCAD forums.
  • The BOLTS issue tracker on GitHub is also a good place to report problems and discuss improvements.
