Motivation

In the recent years, several free and open CAD packages developed. Although using different approaches, they all lacked a useful library of standard parts.

Purpose

BOLTS aims to foster synergies between different projects, by building a common database of technical specification, so that contributions by the community of one tool can directly benefit other tools as well. This will make it simpler for each project to provide a useful and extensive library of standard parts.
